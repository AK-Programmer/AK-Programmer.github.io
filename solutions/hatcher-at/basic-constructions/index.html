Problem 1

Show that composition of paths satisfies the following cancellation property: If \(f_0 g_0 \simeq f_1 g_1\) and \(g_0 \simeq g_1\) then \(f_0 \simeq f_1\).

We first show that \(f_0\) and \(f_1\) have the same endpoints. Since \(f_0 g_0 \simeq f_1 g_1\), \(f_0(0) = f_1(0)\). Since \(g_0 \simeq g_1\), \(g_0(0) = g_1(0)\), and since we're dealing with path compositions, we know that \(f_0(1) = g_0(0)\) and \(f_1(1) = g_1(0)\), which means \(f_0(1) = f_1(1)\).

We must then show that there is a homotopy between \(f_0\) and \(f_1\). Our existing homotopy is \(h: I \times I \to X\), so then the desired homotopy is simply a reparametrization of the restriction \(h|[0, 1/2] \times I\) so that its domain is once again \(I \times I\).

This directly follows from the definition of path composition, where the two composed paths are reparametrized so that they can be 'traversed in half the time', so that traversing both of them takes unit time. As a result, for composable paths \(f\) and \(g\), \(f g (1/2) = f(1)\).

\(\tag*{$\blacksquare$}\)
