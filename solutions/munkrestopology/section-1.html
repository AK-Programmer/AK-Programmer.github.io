Problem 1: Check the distributive laws for \(\cup\) and \(\cap\), and DeMorgan's laws.
Starting with the first distributive law, \[A \cap (B \cup C) = (A \cap B) \cup (A\cap C)\] \[x \in A \cap (B \cup C) \iff x \in A \ \textrm{and} \ (x \in B \ \textrm{or} \ x \in C)\] \[\iff (x \in A \ \textrm{and} \ x \in B) \ \textrm{or} \ (x \in A \ \textrm{and} \ x \in C) \iff (A \cap B) \cup (A\cap C) \ \ \ QED\] Very similar logic applies to the second distributive law, \[A \cup (B \cap C) = (A \cup B) \cap (A\cup C)\] Next, we have DeMorgan's first law (to be honest, I'm not sure what the order is), \[A - (B \cup C) = (A - B)\cup (A-C)\] \[x \in A - (B \cup C) \iff x \in A \ \textrm{and} \ x \not\in B \ \textrm{and} \ x \not\in C\] \[\iff x \in A - B \ \textrm{and} \ x \in A - C \iff (A-B) \cup (A-C) \ \ \ QED\] Once again, similar logic applies to DeMorgan's other law, \[A - (B \cap C) = (A-B) \cup (A - C)\]
